# Open_UnitMate

一个基于 Python 和 PyQt6 开发的现代化单位转换器。

## 主要特性

- 现代化图形界面 (PyQt6)
- 支持多种单位转换（长度、重量、面积、体积等）
- 极轻量级，运行速度快
- 支持系统托盘运行，防止误关闭

## 安装与运行

1. 克隆仓库：

   ```bash
   git clone https://github.com/ustak/Open_UnitMate.git
   cd Open_UnitMate
   ```

2. 安装依赖：

   ```bash
   pip install PyQt6
   ```

3. 运行程序：

   ```bash
   python UnitMate.py
   ```
